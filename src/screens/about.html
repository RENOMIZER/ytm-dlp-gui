YTM-DLP

YTM-DLP is an app created for sole purpose of downloading music off YouTube Music in particular and YouTube in general.
It does not download video, only audio.
It provides convinient way of editing said audio's metadata on the spot.
It was created as a practice project, to try my hand at development of desktop apps.
It is my first attempt at making such, so be aware of bad code and tons of bugs which you can report back to me and I'll try to fix them.

If you are wondering "How to use this app?" you can refer to README file in the GitHub repository.
